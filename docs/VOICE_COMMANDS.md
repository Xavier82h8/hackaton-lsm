# 🎤 UNIVOZ - Comandos de Voz para Usuarios Ciegos

## Activación

### Primera vez
1. La app anunciará: *"UNIVOZ está lista. Para activar comandos de voz, diga 'activar voz' o toque el botón de micrófono."*
2. **Opción A:** Di **"activar voz"**
3. **Opción B:** Toca el botón de micrófono en la esquina superior derecha

### Botón de Comandos de Voz
- Ubicado en la **esquina superior derecha** de las pantallas de Modo Ciego
- Ícono de micrófono 🎤
- Al activarse, escucharás: *"Comandos de voz activados. Puede comenzar a hablar."*

## 💡 Consejos de Uso

### Para Mejor Reconocimiento:
1. **Habla claro** y a un ritmo normal
2. **Evita ruido** ambiental excesivo
3. **Mantén el micrófono** cerca de tu boca
4. **Espera el tono** de confirmación antes de hablar

### Feedback de Audio:
- ✅ **Comando reconocido:** *"Comando ejecutado: [nombre del comando]"*
- ❌ **Comando no reconocido:** *"Comando no reconocido. Diga 'ayuda de comandos' para ver la lista."*
- 🎤 **Escuchando:** *"Escuchando comando..."*
- 🔇 **Sin voz detectada:** *"No se detectó voz. Intente nuevamente."*

## 🔧 Solución de Problemas

### "El reconocimiento de voz no está disponible"
- **Causa:** Tu navegador no soporta la API de reconocimiento de voz
- **Solución:** Usa Google Chrome, Microsoft Edge o Safari

### "No se detectó micrófono"
- **Causa:** El navegador no tiene permiso para usar el micrófono
- **Solución:** 
  1. Verifica que el micrófono esté conectado
  2. Otorga permisos de micrófono al navegador
  3. Revisa la configuración de privacidad

### "Comando no reconocido"
- **Causa:** El comando no está en la lista o hubo ruido ambiental
- **Solución:**
  1. Di *"ayuda de comandos"* para escuchar la lista completa
  2. Intenta nuevamente en un ambiente más silencioso
  3. Habla más claro y pausado

## 🚀 Acceso Rápido

### Desde Cualquier Pantalla de Modo Ciego:
1. **Toca dos veces** la parte superior de la pantalla para activar voz
2. **Di el comando** claramente
3. **Espera confirmación** de audio

### Atajos de Teclado (Futuro):
- `Ctrl + M`: Activar/desactivar micrófono
- `Ctrl + H`: Ir a inicio
- `Ctrl + D`: Ir a dictar
- `Ctrl + A`: Abrir ajustes
